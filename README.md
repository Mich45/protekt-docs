# Protekt documentation

Welcome to the Protekt documentation site. This repository contains the source for the public docs for Protekt, a developer-friendly authentication platform.

## Quickstart

### Prerequisites

- Node.js 18 or later
- Yarn installed

### 1. Clone the repository

```bash
git clone https://github.com/Mich45/protekt-docs.git
cd protekt-docs
```

### 2. Install dependencies

```bash
yarn install
```

### 3. Start the development server

```bash
yarn start
```

Open the local site in your browser at `http://localhost:3000`.

### 4. Build for production

```bash
yarn build
```

### 5. Deploy

```bash
yarn deploy
```

This will publish the site using the repository's configured GitHub Pages deployment.
